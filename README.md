# FileEncryptAndDecrypt
使用Java编写的一个文件加解密ww~

例子喵~
文件夹加密:
  java -jar ".../FileEncrypt.jar" "D:\Example"
单文件加密:
  java -jar ".../FileEncrypt.jar" "D:\Example\Test.exe"
文件夹加密:
  java -jar ".../FileDecrypt.jar" "D:\Example"
单文件加密:
  java -jar ".../FileDecrypt.jar" "D:\Example\Test.exe"
